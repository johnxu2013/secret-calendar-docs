Secret Calendar Nav

Your Secret Calendar and Navigation Experience on iOS

ℹ️ About Secret Calendar Nav

Secret Calendar Nav is a privacy-focused calendar application for iOS. Built with modern Apple technologies, it provides security to manage your events, tasks, and schedules while keeping your data completely private on your device.

If the physical device facing the risk of owner losting control, he/she can click the "Delete All Events" link to remove all events, avoiding potential troubles. The link is located at Settings -> Manage Data page. There is no confirmation of delete (for quick action). So please "Export Events" to safe storage for backup before starting a risky travel or similar event.

The app combines the simplicity of traditional calendaring with powerful features like recurring events, voice notes, smart notifications, and integration with your iPhone's built-in Calendar app.

✨ Key Features

🔒 Complete Secret

All data stored locally on your device. It reads from public calendars, but keep secret calendar events locally without transfer to cloud. no tracking.

📝 Map Navigation

Events can add map and navigation. For example, putting recurring events with map navigation for planning work onsite

📝 Event Management

Create, edit, and organize events with detailed information including location, notes, and categories.

🔄 Recurring Events

Support for daily, weekly, bi-weekly, monthly, and yearly recurring events with flexible completion tracking.

🎤 Voice Notes

Attach voice recordings to events for quick capture of thoughts and details.

⏰ Smart Alarms

Customizable daily alarm notifications to remind you of upcoming and incomplete tasks.

🔔 Event Reminders

Flexible reminder system with options from event time to weeks in advance.

📱 iPhone Calendar Integration

View events from your device calendars alongside Secret Calendar events in one unified view.

🏷️ Categories

Organize events with custom categories, each with its own color coding and visibility controls.

✅ Task Completion

Mark events as complete with optional completion notes, including support for individual recurring event instances.

🔍 Search & Filter

Powerful search functionality to quickly find events by title, location, or notes.

📍 One Click Delete

In some extreame situations, you can use one click to delete all events.

⚙️ Technical Details

System Requirements

  • iOS 17.0 or later
  • iPhone compatible
  • Notifications permission for reminders and alarms
  • Secret Calendar events are not shared with other apps
  • Microphone permission for voice notes (optional)

🔐 Privacy & Security

Secret Calendar Nav is designed with privacy as a core principle:

  • No Cloud Sync: All data stays on your device
  • No Analytics: We don't collect usage data or track user behavior
  • No Account Required: Start using immediately without sign-up
  • Local Storage Only: Events, notes, and voice recordings never leave your device
  • Calendar Access: Device calendar integration is optional and read-only
  • Minimal Permissions: Only requests permissions for features you actually use

For complete privacy details, please see our Privacy Policy.
